# 🤖 School Diary Automation Bot

Телеграм-бот на **Python** для автоматизации работы с электронным дневником (Сетевой Город). 

## ✨ Что умеет бот:
- **Парсинг ДЗ**: Заходит под логином/паролем, делает скриншот страницы и отправляет в чат.
- **Умное расписание**: По команде `/time` считает время до конца урока или перемены.
- **Автозапуск**: В комплекте идут скрипты для скрытого запуска в фоне на Windows.
- **Планировщик**: Работает по расписанию (отправка ДЗ утром и вечером).

## 🛠 Стек технологий:
- **Язык**: Python 3.11+
- **Библиотеки**: Aiogram 3.x, Playwright (Chromium), APScheduler.
- **Автоматизация**: VBScript и Batch для фоновой работы.

## 🚀 Как использовать:
1. Установите зависимости: `pip install -r requirements.txt`
2. Установите браузер: `playwright install chromium`
3. Настройте свои данные в `config.py`.
4. Для запуска в фоновом режиме используйте `invisible.vbs`.
